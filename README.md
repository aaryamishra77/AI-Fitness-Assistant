# 🏋️ AI Fitness Assistant

An AI-powered fitness assistant that generates personalized workout recommendations based on user goals, fitness level, and preferences. The application combines a modern React frontend with a Python backend and Large Language Models (LLMs) to deliver intelligent fitness guidance.

## 🚀 Features

- 🤖 AI-generated personalized workout plans
- 💪 Exercise recommendations based on user goals
- 🧠 LLM-powered fitness guidance
- 🎯 Beginner to advanced workout suggestions
- 📱 Responsive and user-friendly interface
- ⚡ Fast React frontend with Vite
- 🔒 Backend API for handling AI requests

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- JavaScript

### Backend
- Python
- Flask/FastAPI
- REST API

### AI
- Large Language Models (LLMs)
- Prompt Engineering
